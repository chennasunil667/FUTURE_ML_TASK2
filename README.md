# Support Ticket Classification

## Overview
This project automatically classifies customer support tickets and assigns priority levels using Natural Language Processing (NLP) techniques.

## Technologies Used
- Python
- NLTK / spaCy
- Scikit-Learn
- Pandas
- Jupyter Notebook

## Features
- Text Cleaning and Tokenization
- Ticket Category Classification
- Priority Prediction
- Model Evaluation
